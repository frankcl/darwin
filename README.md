# darwin
爬虫系统：达尔文树皮蜘蛛，能结出世界上最大最牢固的网

## 特点

* 支持自定义脚本，针对特定站点定制抽链和结构化逻辑
* 支持域名/站点粒度抓取调度：具备礼貌性控制，管理站点/域名粒度单位时间并发度
* 支持文本（HTML、JSON等）、资源（图片和视频）、流媒体（M3U8流）等数据抓取
* 多层次爬虫数据管理：应用->计划->任务->抓取记录
